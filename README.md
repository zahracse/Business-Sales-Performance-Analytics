# Business-Sales-Performance-Analytics
Business Sales Performance Analytics project focused on revenue trends, product performance, regional profitability, and data-driven business recommendations using real-world sales data.
📌 Project Overview
This project analyzes business sales data to identify revenue trends, high-performing products, and profitable regions.

🎯 Objective

To help the business make data-driven decisions using sales performance insights.
🛠 Tools Used
Power BI
Excel
Data Cleaning Techniques

📊 Key Insights
Technology category drives highest revenue.
West region shows strongest profitability.
Sales peak in Q4 seasonally.

🚀 Recommendations
Increase investment in top-performing products.
Expand marketing in profitable regions.
Review pricing for low-margin categories.
